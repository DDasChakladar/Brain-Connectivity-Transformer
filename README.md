# Brain-Connectivity-Transformer
This repository provides the implementation of Brain Connectivity Transformer (BCT), a graph-based transformer framework for multi-task learning on fMRI brain connectivity data. <br>
This code is a TensorFlow implementation of our paper "Multi-Task Brain Connectivity Transformer for
Joint Cognitive Task and Disorder Classification".<br>
The code is written in Google Colab platform. Change the folder paths in the code as per requirement.
